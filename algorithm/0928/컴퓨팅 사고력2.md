## 기초수식

- 문제8 : T(n) = T(n-1) + 1/n

- sol) T(1) = 1, T(1) = T(0) + 1 이므로 T(0) = 0

  T(n) = T(n-1) + 1/n n

  ​        = T(n-2) + 1/(n-1) + 1/n 

  ​        = T(n-3) + 1/(n-2) + 1/(n-1) + 1/n

  ​        = T(n-n) + 1/{n-(n-1)} + 1/{n-(n-2)} + ... + 1/(n-1) + 1/n

  ​		= 1 + 1/2 + ... + 1/(n-1) + 1/n

  ​		>=  1 + 1/2 + (1/4 + 1/4) + (1/8+ + 1/8 + 1/8 + 1/8) + ... + 1/n
  
  ​		= nlogn
  
  따라서 **O(nlogn)**
